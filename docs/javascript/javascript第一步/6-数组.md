
# 数组
包含多个值的对象

## 创建数组
用方括号构成，其中包含用逗号分隔的元素列表

```javascript
let shopping = ["bread", "milk", "cheese"];
```

## 访问和修改数组元素
```javascript
let shopping = ["bread", "milk", "cheese"];
console.log(shopping[0]); //bread

shopping[0] = "tahini";
console.log(shopping) // ['tahini', 'milk', 'cheese']
```

## 获取数组长度
```javascript
let shopping = ["bread", "milk", "cheese"];
console.log(shopping.length) //3
```
